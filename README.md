**本项目使用正点原子硬件与基础代码编写**

当前编辑版本与编译环境：

- Windows 7
- STM32F103ZET6
- Keil MDK 5.14
- STemWin5.26
- uCOS III

# 先将目录中的WZTSYSTEM文件夹复制到SD卡根目录，并运行下载程序.
